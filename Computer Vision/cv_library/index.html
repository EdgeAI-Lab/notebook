    open source computer vision librarys¶

    1.OpenCV¶

    OpenCV（开源计算机视觉库）是在BSD许可下发布的，因此它可以免费用于学术和商业用途。 它具有C ++，Python和Java接口，支持Windows，Linux，Mac OS，iOS和Android。 OpenCV专为提高计算效率而设计，专注于实时应用。该库以优化的C / C ++编写，可以利用多核处理。 通过OpenCL启用，它可以利用底层异构计算平台的硬件加速。
