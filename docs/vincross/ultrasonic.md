## 2. 波形测量

### 2.1 整体波形

![](../assets/images/vincross/ultrasonic/ultrasonic_waveform_00.jpg)

* 蓝色是发出的8个波形，发出的频率是40K HZ
* 粉红色是ECHO引脚的输出
* 青色是P00超声波接收原始输出，接收的频率也是40K HZ
* 能量越大，接收端波形个数越多
