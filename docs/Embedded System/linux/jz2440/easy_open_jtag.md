# EOP(Easy Open JTAG)

EOP百问科技出品的调试器，用于下载裸机程序。

EOP的驱动在Windows10下驱动安装十分麻烦，不想折腾可以在Ubuntu上使用EOP（免驱）。

[EOP for Ubuntu下载地址](https://pan.baidu.com/s/1Yz_5x_PRXMeIoEhU8I2G-g)
