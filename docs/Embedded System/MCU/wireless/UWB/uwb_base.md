* 虽然UWB的频率是从3 ~ 10.6GHz，但是其中5.8GHz是留给移动5G的（图中白色部分），并且通道5和6在日本和韩国已经被占用，所以未来在汽车和IOT领域，UWB很可能将会较多的使用通道8和通道9


* 在IEEE802.15.4z规范中，UWB主要使用CH5、CH6、CH8、CH9
