# UWB定位算法

## 1. 测距方法

UWB的定位是基于距离值计算得到的。常用的UWB测距方法有：

* SS-TWR
* DS-TWR
* TDOA
* AOA


## 1. 基本定位算法

根据距离值求解标签位置。
